About This Project
ABOUT SPORTSPOT

A BWD coursework project by CVM University students.

SportSpot is a Sports Slot Booking Web Application developed as part of the Basic Web Design (BWD) coursework at CVM University (Charutar Vidya Mandal).

The platform enables users to browse available sports, view slot availability in real-time, and complete bookings — all through a clean, dark-themed interface built with HTML5, CSS3, JavaScript, and Bootstrap 5.

This project demonstrates multi-page website structure, consistent UI/UX design principles, responsive layouts, and client-side interactivity as covered in the BWD syllabus (40 hours total across 7 units).

HTML5 CSS3 JavaScript (ES6) Bootstrap 5 Flexbox CSS Grid Responsive Design DOM Events
